## 这是什么？

通过云端部署 node 和 hexo 框架的方式，实现 hexo 在云端自动生成静态文件和自动发布。

适合自己没有 vps ，又需要随时随地写博客并发布的用户；或者需要在多台不同设备上使用 hexo 写博客，却不想在这些设备上都安装 hexo 的用户。
